## How to Use These Examples

1. Copy the match data sections for each field in the analyzer
2. Paste multi-line data (Events, Commentary, Statistics) into respective textarea fields
3. Select analysis type (Complete, Summary, Tactics, or Pivots)
4. Choose your preferred style and focus area
5. Click "Analyze Match"

Each example represents different game genres:
- **VALORANT**: Tactical FPS with abilities
- **Clash of Clans**: Strategy/Tower Defense
- **PUBG Mobile**: Battle Royale
- **League of Legends**: MOBA
- **CS:GO**: Pure tactical FPS

The AI model will provide strategic analysis tailored to each game's mechanics!
